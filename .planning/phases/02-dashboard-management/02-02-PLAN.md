**In public/js/studio-canvas.js -- render() outline logic:**

4. Replace the existing outline assignment with multi-select-aware logic:
   - If `app.selectedWidgetIds && app.selectedWidgetIds.size >= 2 && app.selectedWidgetIds.has(wc.id)`: set `card.style.outline = '2px dashed #60A5FA'` and `card.style.boxShadow = '0 0 0 2px rgba(96,165,250,0.10)'`
   - Else if `wc.id === app.selectedWidgetId`: set `card.style.outline = '2px solid #FDA4D4'` and `card.style.boxShadow = ''`
   - Else: set `card.style.outline = '2px solid transparent'` and `card.style.boxShadow = ''`

**In public/js/studio-canvas.js -- rubber-band selection:**

6. Replace the existing `page.addEventListener('click', ...)` handler that deselects on empty space with a `mousedown`-based rubber-band system. Implementation:
   - On `mousedown` on `page`: check `e.target.closest('.widget')` -- if truthy, return. Check for resize handles too.
   - Record `rbStartX = e.clientX`, `rbStartY = e.clientY`
   - On `mousemove` (attached to `document`): if delta < 4px both axes, skip. Otherwise create a `div` with className `rubber-band-rect`, append to `page`, set `position:absolute`, `pointer-events:none`, `z-index:50`. Compute rect from min/max of start and current coords, subtract `page.getBoundingClientRect()` offset. Update left/top/width/height.
   - On `mouseup` (attached to `document`): remove rubber-band element. Remove mousemove/mouseup listeners. If no significant movement: treat as deselect click -- clear `app.selectedWidgetIds`, clear `app.selectedWidgetId`, set all widget outlines to transparent, show dashboard props/placeholder. If movement occurred: compute intersection -- for each `.widget` card in page, compare `getBoundingClientRect()` against selection rect bounds (left < right && right > left && top < bottom && bottom > top). Populate `app.selectedWidgetIds` with matching `card.dataset.widgetId` values. Sync `app.selectedWidgetId`. Update outlines. If 2+ selected, call `app.showMultiSelectProps()`. If 1, call `app.showWidgetProps(id)`.

**Ensure `dataset.widgetId` is set on cards:**

7. In the `render()` function where widget cards are created, add `card.dataset.widgetId = wc.id;` if not already present.
