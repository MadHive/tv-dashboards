## Interaction Contracts

### Wizard Open/Close
- Trigger: click `+` button in sidebar dashboards section header.
- Open: `style.display = 'flex'` on `.studio-modal` (existing pattern).
- Close: backdrop click, Escape key, or modal-close button â€” all call `closeWizard()`.
- Keyboard: Tab cycles through step fields. Enter on last field of Step 1 advances to Step 2. Enter on Step 2 submits.

### Multi-select: Shift+click
- First click: selects single widget (existing `selectedWidgetId`, sets `selectedWidgetIds = new Set([id])`).
- Shift+click on unselected widget: adds to `selectedWidgetIds`.
- Shift+click on selected widget: removes from `selectedWidgetIds` (toggle).
- If `selectedWidgetIds.size === 1`, treat as single-select (show full properties panel).

### Multi-select: Rubber-band
- mousedown on canvas empty space (no widget at cursor): begin rubber-band drag.
- mousemove: update rect dimensions, draw overlay.
- mouseup: compute widget intersection, populate `selectedWidgetIds`. Minimum drag distance: 4px (avoid accidental trigger on click).
- Rubber-band and widget drag are mutually exclusive â€” widget drag begins only when mousedown lands on a widget.

### Ctrl+C / Ctrl+V
- Ctrl+C: deep-copy all widgets in `selectedWidgetIds` to `_widgetClipboard`. Show clipboard indicator. Do not clear selection.
- Ctrl+V: if `_widgetClipboard.length > 0`, paste into active dashboard. Regenerate widget IDs. Attempt original positions â†’ `StudioCanvas.snapToNearest()` for each. Mark dirty. Re-render. Select pasted widgets. Clear clipboard.
- Ctrl+V with empty clipboard: no-op (no toast, no error).

### Dashboard Duplication
- Click â§‰ duplicate button on nav item.
- Call `POST /api/dashboards/:id/duplicate`.
- On success: `loadConfig()` + `renderSidebar()`. Auto-select copy. Trigger inline rename on the new item.
- On error: `showToast("Duplication failed â€” try again", "error")`.
